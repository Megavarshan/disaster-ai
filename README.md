# Decision-Admissibility Disaster Intelligence Platform (DADIP)

**DADIP** is a next-generation, AI-powered disaster management ecosystem built to modernize how government agencies and emergency responders (such as the NDRF and SDRF) handle crises. 

In high-stakes emergency scenarios, operations centers are often overwhelmed by fragmented data, unverified citizen reports, and static dashboards. DADIP solves this by serving as a centralized, highly intelligent command center that bridges the gap between raw data and actionable deployment.

### Key Capabilities & Features

*   **AURA AI Risk & Anomaly Assessment:** The platform is powered by AURA, an integrated AI assistant that continuously analyzes incoming data streams. AURA automatically assesses risk severity, verifies the admissibility of citizen reports against historical data, and flags anomalies, completely eliminating the "noise" that traditionally slows down emergency response.
*   **Multi-Modal Data Pipeline:** DADIP seamlessly ingests data from multiple channels—including live citizen incident reports, localized weather nodes, and custom CSV dataset uploads. The pipeline instantly parses this data to update the live operational map.
*   **Dynamic Resource Orchestration:** Built specifically for government operations, the platform allows commanders to view real-time capacities of help centers and instantly deploy specialized teams (NDRF/SDRF) directly to verified incident hotspots.
*   **Frictionless Official Reporting:** The platform features a robust, custom-engineered PDF generation engine. With a single click, operations officers can instantly download pixel-perfect, highly styled Official PDF Reports (Daily Situation Reports, Incident Summaries, and Risk Assessments) completely offline and directly to their local machine, ensuring critical data is always portable.
*   **Modern, High-Performance Interface:** Built on Next.js and styled with the latest Tailwind CSS v4, the dashboard features a gorgeous, dark-mode, glassmorphic UI. This ensures that the platform is not only exceptionally fast and responsive but also provides high cognitive clarity for operators working under extreme pressure.

**The ultimate goal of DADIP is simple:** To replace reactive chaos with proactive, data-admissible intelligence, ensuring that life-saving resources get to exactly where they are needed, faster than ever before.

## Getting Started

First, install dependencies:
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the platform in action.
